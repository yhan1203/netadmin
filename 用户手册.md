# 用户手册 — netadmin

## 快速安装

```bash
pip install netadmin
```

验证安装：

```bash
netadmin --version
# netadmin, version 0.1.0
```

## 配置设备清单

创建 `config.yaml`：

```yaml
devices:
  - host: 192.168.1.1
    name: core-switch
    vendor: huawei
    device_type: huawei
    username: admin
    password: ""          # 留空则运行时提示

  - host: 192.168.1.2
    name: access-switch
    vendor: cisco
    device_type: cisco_ios
    username: admin
    password: ""

defaults:
  username: admin
  password: ""
  port: 22
  timeout: 30
```

> **安全提示：** 密码支持通过环境变量传入，避免明文写入 YAML：
> ```bash
> export NETADMIN_PASS_192_168_1_1=MySecret
> ```

## 常用操作

### 每日巡检

```bash
# 全量巡检所有设备
netadmin check --all

# 单台设备
netadmin check 192.168.1.1
```

### 配置备份

```bash
# 备份所有设备
netadmin backup run

# 查看备份历史
netadmin backup list

# 对比两次备份
netadmin backup diff 1 2
```

### 照猫画虎（核心功能）

场景：5 台新交换机上线，需要和 A 机配置一致。

```bash
# 1. 从 A 学配置
netadmin learn 192.168.1.1 -o template.yaml

# 2. 查看模板（YAML 格式）
cat template.yaml

# 3. 修改模板中的占位符（IP、名称等）
vim template.yaml

# 4. 试运行看效果
netadmin apply template.yaml -d 192.168.1.2 --dry-run

# 5. 确认无误后正式部署
netadmin apply template.yaml -d 192.168.1.2

# 6. 批量部署
netadmin apply template.yaml -d 192.168.1.2,192.168.1.3,192.168.1.4,192.168.1.5
```

### 安全审计

```bash
# 审计单台
netadmin audit 192.168.1.1

# 审计全部
netadmin audit --all

# 输出：安全评分 + 逐项检查结果
```

### 设备发现（交接专用）

```bash
# 扫描网段
netadmin scan 192.168.1.0/24
```

### VLAN 管理

```bash
# 查看 VLAN
netadmin vlan list 192.168.1.1

# 创建 VLAN
netadmin vlan create 192.168.1.1 10 office

# 删除 VLAN
netadmin vlan delete 192.168.1.1 99

# 端口分配
netadmin vlan assign 192.168.1.1 GigabitEthernet0/0/1 10
netadmin vlan assign 192.168.1.1 GigabitEthernet0/0/2 10 --mode trunk
```

### 接口状态

```bash
# 查看所有接口
netadmin interface list 192.168.1.1

# 查看单个接口详情
netadmin interface detail 192.168.1.1 GigabitEthernet0/0/1
```

### 批量执行

```bash
# 在所有设备上执行命令
netadmin exec-all "display vlan"

# 在指定设备执行
netadmin exec 192.168.1.1 "display logbuffer | tail -20"
```

## 配置对比

```bash
# 对比两次备份
netadmin backup diff 1 2

# 查看备份内容
netadmin backup restore 1
```

## 常见问题

### Q: 连接超时怎么办？
A: 检查 SSH 是否开启、IP 是否正确，`netadmin connect` 可单独测试连接。

### Q: 设备不在 config.yaml 里怎么操作？
A: 直接用 `-u` `-p` 参数临时指定：

```bash
netadmin connect 10.0.0.1 -u admin -p MyPass -t huawei
```

### Q: 支持哪些设备？
A: 华为 S/CE 系列（`huawei`），思科 Catalyst/IOS-XE（`cisco_ios`）。理论上所有 Netmiko 支持的设备类型都可用，传 `--device-type` 参数即可。

### Q: 密码能加密吗？
A: 支持环境变量 `NETADMIN_PASS_<IP>`，避免 YAML 明文。正式使用建议配合密钥管理工具。

### Q: 备份存在哪里？
A: 默认存在 `./backups/` 目录，SQLite 数据库管理版本索引。
